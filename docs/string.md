# String

> 字符串集

## randomId

> 随记Id, 生成10-12位不等的字符串

```js
randomId(); // 随记Id
```

## randomColor

> 随记生成颜色

* @name randomColor
* @link https://www.jianshu.com/p/54fc0fce46cc
* @param {string} [type='16#'] (16#, rgba, 'hsla')
* @param {number} [alpha=1] 透明度

```js
randomColor("rgba", 0.8); // 随记color
```
